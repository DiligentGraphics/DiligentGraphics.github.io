float Diligent::ComputeMipLevelAttribs::AlphaCutoff = 0

Alpha cutoff value.

When AlphaCutoff is not 0, alpha channel is remapped as follows: 

A_new = max(A_old; 1/3 * A_old + 2/3 * AlphaCutoff)
