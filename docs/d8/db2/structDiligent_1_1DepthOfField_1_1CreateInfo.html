bool Diligent::DepthOfField::CreateInfo::EnableAsyncCreation = false

Whether to enable asynchronous shader and pipeline state creation.

If enabled, the shaders and pipeline state objects will be created using the engine's asynchronous creation mechanism. While shaders are being compiled, the effect will do nothing and return the input color.
