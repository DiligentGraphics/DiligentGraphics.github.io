virtual ID3D11Resource *DILIGENT_CALL_TYPE Diligent::ITextureD3D11::GetD3D11Texture ( ) const
pure virtual

Returns a pointer to the ID3D11Resource interface of the internal Direct3D11 object.

The method does NOT increment the reference counter of the returned object, so Release() must not be called.
