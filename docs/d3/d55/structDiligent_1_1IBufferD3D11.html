virtual ID3D11Buffer *DILIGENT_CALL_TYPE Diligent::IBufferD3D11::GetD3D11Buffer ( ) const
pure virtual

Returns a pointer to the ID3D11Buffer interface of the internal Direct3D11 object.

The method does NOT increment the reference counter of the returned object, so Release() must not be called.
