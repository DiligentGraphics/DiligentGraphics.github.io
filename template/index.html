Diligent Engine

Diligent Engine is a high-performance, lightweight graphics API abstraction layer and rendering framework designed for seamless cross-platform development. It provides a unified interface that leverages the power of modern graphics APIs such as Direct3D12, Vulkan, Metal, and WebGPU, while also maintaining robust support for legacy platforms through Direct3D11, OpenGL, OpenGLES, and WebGL.
Diligent Engine offers a single, consistent front-end API and utilizes HLSL as its universal shading language across all platforms and rendering back-ends. Additionally, it supports platform-specific shader formats, including GLSL, MSL, DirectX bytecode, and SPIR-V, enabling developers to optimize graphics performance on each targeted system.
Engineered for versatility, Diligent Engine serves as an ideal graphics subsystem for integration into game engines, interactive simulations, and any other 3D visualization applications. It is open-source and distributed under the permissive Apache 2.0 license, ensuring that developers can freely use and adapt the engine within their projects.
